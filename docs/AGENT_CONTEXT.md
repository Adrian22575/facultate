# Agent Context

Acest fisier ramane ca pointer scurt pentru compatibilitate cu agentii mai vechi.

Contextul actual al proiectului este in:

- `AGENTS.md`
- `docs/agent-repo-map.md`
- `docs/agent-playbook.md`
- `docs/agent-lessons.md`

Aplicatia activa este Next.js App Router. Versiunea statica veche a fost arhivata in `backup/`, iar fisierele HTML legacy nu mai sunt sursa principala.

Pentru audit rapid al structurii:

```powershell
npm run workspace:audit
```
