# Academic Seed Sources

Acest fișier documentează sursele folosite pentru lista inițială de instituții introdusă prin `0006_seed_academic_institutions.sql`.

## Universități

Baza pentru universități pornește din sursele oficiale ale Ministerului Educației:

- `Lista universităților din România` pe `edu.ro`
- paginile și listele oficiale despre instituții de învățământ superior acreditate

În seed am inclus un subset practic, orientat pe centre universitare mari, ca punct de pornire pentru onboarding.

## Licee / școli

Pentru preuniversitar, referința folosită este setul public al Ministerului Educației disponibil pe `data.gov.ro`:

- `Coordonate geografice unități de învățământ (geocodare)`

Lista de licee din seed este intenționat un starter set redus, orientat pe câteva orașe mari și colegii cunoscute, nu o rețea exhaustivă.

## Notă de produs

- seed-ul nu trebuie tratat ca nomenclator complet național
- utilizatorii pot adăuga instituții lipsă direct din onboarding
- pe termen mediu, lista ar trebui extinsă prin import controlat din surse oficiale mai complete
